# shocked-pikachu
 inserts a pikachu asci art into your react component

## Install

```
$ npm install react-shocked-pikachu
```

## Usage
```js
import React from 'react';
import Pikachu from 'react-shocked-pikachu';

const App = () => {
    return (
        <Pikachu/>
    )
}

```
